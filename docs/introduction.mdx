## What's New in v0.3.0

<Accordion title="Enhanced Wallet Management">
  Complete wallet operations with controller/custodian administration, address book functionality, and cycle transfer capabilities.
</Accordion>

<Accordion title="NNS Canister Support">
  Direct integration with Network Nervous System canisters including governance, ledger, and other system canisters with specialized query interfaces.
</Accordion>

<Accordion title="Advanced Filtering">
  Powerful filtering system for managing large numbers of canisters by type, network, and search criteria.
</Accordion>

<Accordion title="Cache Management">
  Intelligent DFX version management with installation, removal, and cache monitoring capabilities.
</Accordion>

<Accordion title="Shell Completion">
  One-click setup for dfx shell completion across bash, zsh, and fish shells.
</Accordion>
